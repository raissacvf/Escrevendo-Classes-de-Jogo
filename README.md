Escrevendo as Classes de um Jogo ⚔️

Terceiro desafio do Bootcamp de Lógica de Programação do Santander. O objetivo foi consolidar os conhecimentos em Orientação a Objetos criando uma classe genérica para representar heróis de uma aventura.

 O Desafio:

Criar uma classe `Heroi` que possua propriedades como nome, idade e tipo. Além disso, implementar um método `atacar()` que exibe uma mensagem baseada na classe do herói, seguindo a tabela:

| Tipo      | Ataque                    |
|-----------|---------------------------|
| Mago      | Usou magia                |
| Guerreiro | Usou espada               |
| Monge     | Usou artes marciais       |
| Ninja     | Usou shuriken             |

-> Estruturas Utilizadas

- **Classes e Objetos:** Criação da classe `Heroi` e instanciação dos objetos.
- **Métodos:** Função `atacar()` dentro da classe.
- **Estruturas de Decisão:** `switch/case` para verificar o tipo do herói.
- **Laços de Repetição:** `for` para iterar sobre uma lista de heróis criados.
- **Variáveis e Operadores:** Manipulação de dados e concatenação de strings.

-> Como Executar
- Clone o repositório.
- Abra o terminal na pasta do projeto.
- Execute node index.js (certifique-se de ter o Node.js instalado) ou coloque no site playcode.io.
